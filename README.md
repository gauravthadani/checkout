# Checkout
A checkout system for a computer store

### Tech

Checkout is written in golang with IntelliJ IDEA

### Installation of Golang

https://golang.org/doc/install


### Running Tests 

clone at ~/src/github.com/gauravthadani/checkout

```sh
cd checkout
go test ./... -v
```

### Running 

clone at ~/src/github.com/gauravthadani/checkout

```sh
cd checkout
go run main.go
```
